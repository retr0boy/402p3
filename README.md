# 402p3
ROBT 402 Project 3

PWM controlling DC motor with encoder using AltIMU-10 v5

- Task 1 is just spinning motor at different duty, frequency and directions.
- Task 2 is attaching encoder and retrieving values from it
- Task 3 is PD control of motor's angle using values from IMU sensor
SF will throw unexpected errors, so disconnect it or tie to GND (it is highly unrecommended but it is what it is)
